## 当时为什么提出这条路线

分类别结果显示，固定类别—月份物候曲线与主要误差来源并不完全匹配：

- 季节性明确且现有结果较好的类别，额外日历知识空间有限；
- 聚合/混合类别存在显著类内异质性，单一曲线代表性不足；
- 多年生与长期覆盖类别可能更依赖空间纹理、地块结构和管理方式；
- 少数类别还可能受到样本量、类别不平衡和近邻类别混淆影响。

因此旧方案曾建议以困难类别诊断为主线，再依次尝试类别均衡、类别感知采样、层级监督、多时序原型和空间先验。

## 为什么不再作为主线

助教在 2026-07-17 明确了新的研究目标：现有基线已经足够，后半阶段应研究**通用的异构先验知识注入方法**。困难类别优化可以提高局部分数，但不能独立回答“异构先验怎样进入视觉网络”这一核心问题。

因此以下事项不再按旧优先级展开：

- 不以 class-balanced loss 或 sampler 作为下一项核心创新；
- 不因某类 IoU 较低就立即为其设计专用 decoder；
- 不把多原型、层级头和空间正则一次性叠加；
- 不用分类别优化替代通用先验编码与融合机制。

## 仍然保留的可用内容

这条旧路线仍可作为**诊断工具箱**，服务于新方法的任务化分析：

1. 统计 train folds 的类别像素、patch 和地块支持度；
2. 输出原始及归一化混淆矩阵；
3. 将类别分为季节性、多年生和聚合/混合组；
4. 检查提升来自内部区域、边界还是少数样本；
5. 若 CA-HPI 结果指向明确瓶颈，再把多原型或空间知识作为先验 token 的扩展，而不是另一条总线。
